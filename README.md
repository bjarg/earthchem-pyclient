# Earthchem & PyData

Test status: [![Coverage Status](https://coveralls.io/repos/github/jesserobertson/earthchem-pyclient/badge.svg?branch=master)](https://coveralls.io/github/jesserobertson/earthchem-pyclient?branch=master) [![Build Status](https://travis-ci.org/jesserobertson/earthchem-pyclient.svg?branch=master)](https://travis-ci.org/jesserobertson/earthchem-pyclient)

This project wraps the Earthchem web services to provide easy access to geochemical data from [IEDA](https://www.iedadata.org/) in ready-to-use format in your favourite PyData environment.

Maintainer: Jess Robertson (jesse.robertson _at_ csiro.au)
